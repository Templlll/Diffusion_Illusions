Creating Visual Cognitive Illusions

Computer Vision Project

Bowen Tan, Shi Sheng, Hanyu Zheng
Peking University
Report PDF Code

Abstract

This project explores the computational generation of multi-arrangement visual illusions by leveraging the rich generative priors of pre-trained text-to-image diffusion models. Our goal is to reproduce and extend a unified gen erative framework capable of automating this process.

We systematically implement five distinct types of illusions. Furthermore, we aim to advance beyond text-only conditioning by introducing "Image Prompt Supporting" and enable more precise control over the visual cognitive experience.

Task 1: Flippy Illusions

We reproduced the generative framework for orientation-dependent optical illusions (180-degree rotation). The images manifest distinct semantics upon rotation.


Task 2: Rotation Overlays

Investigates the geometric entanglement of two image layers via pixel-wise multiplication, allowing semantics to change upon 90¬∞ rotation.

Extending the Framework

We jointly use both text prompt and image prompt to generate illusions, successfully embedding image prompts (e.g., Mona Lisa, Lion) into scenes described by texts.
